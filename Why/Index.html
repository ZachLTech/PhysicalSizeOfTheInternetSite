Why make this?

Well, the somewhat basic thought came into my mind one day while bored and thinking outloud with a few friends: what would it be like if someone were able to download the entire internet on only local drives? Then I wondered things such as: how much physical space would it take? and how much it would cost? So I did a few google searches and couldn't find anything that estimated this and decided to calculate it myself. After some basic data hunting and math, I was able to calculate a (somewhat inaccurate and estimated) size of the internet. Then, it hit me- What if I automated this task and turned it into a website for other curious people that, for some crazy unknown reason, might think to themselves... "I wonder how big the internet really is (physically)". Thats when I went on my way to code my heart out in this site. I decided to make it do more than just tell the physical size of the internet; It's also interactive, with the ability to change the units of measure and amount of storage mediums stacked in the stats pages (which unsurprisingly involved a lot of math). This started as a project involving me wanting to create a website with some interesting information, but quickly turned into a journey full of learning as I experimented with many ways to write and implement HTML, CSS, and Javascript building my skills by playing around with different styles, designs, animations, and interactivity in each page.
